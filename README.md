# SVG Asset Manager

A desktop application for browsing and editing SVG files.

## Installation

1. **Install Rust**
   - Go to https://rustup.rs/
   - Follow the installation instructions for your OS
   - Restart your terminal after installation

2. **Clone and Run**
```bash
   git clone <your-repo-url>
   cd <project-name>
   cargo run
```
